Formulario de Log In con validaciones de campos, que lee que el formato de mail ingresado sea valido,
valida que la contraseña sea mayor a 6 caracteres y a su vez que las contraseñas coincidan en los dos 
campos.
Una vez los datos ingresados son correctos, el navegador entrega un mensaje de bienvenida, tambien se 
guarda el mail de forma local y se crea un JSON con el mail y el password que se imprimen por consola.
